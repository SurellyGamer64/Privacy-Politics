Esta política explica qué información puede recopilar esta aplicación, cómo se utiliza y cómo se protege. Al utilizarla, aceptas esta Política de Privacidad.

// 01

Información Recopilada

Para permitir el funcionamiento del juego, esta aplicación puede almacenar la siguiente información:

ID de usuario Nombre visible Progreso Inventario Figuras Estadísticas Nivel Monedas virtuales Config. del juego

No se recopilan datos innecesarios fuera del funcionamiento normal del servicio.

// 02

Uso de la Información

La información almacenada se utiliza únicamente para:

  • Guardar progreso y gestionar inventarios
  • Mostrar estadísticas del juego
  • Mantener las funciones e interacciones del bot
  • Mejorar la experiencia del usuario
  • Corregir errores y problemas técnicos
// 03

Compartición de Información

La información almacenada no se vende ni se comparte con terceros. Solo podrá utilizarse cuando sea necesario para el funcionamiento técnico del servicio.

Tus datos permanecen dentro del sistema. Sin publicidad. Sin venta de datos.
// 04

Seguridad de los Datos

Se realizan esfuerzos razonables para proteger la información almacenada. 🗣🔥

Sin embargo:

  • Ningún sistema es completamente inmune a errores.
  • No se garantiza protección absoluta contra pérdidas inesperadas.
// 05

Conservación de Datos

Los datos pueden mantenerse mientras el servicio exista o mientras sean necesarios para el funcionamiento del juego. Algunas actualizaciones podrían modificar, reiniciar o eliminar datos cuando sea necesario.

// 06

Derechos del Usuario

Los usuarios pueden solicitar:

  • Información general sobre los datos almacenados
  • Corrección de errores relacionados con sus datos
  • Eliminación de información cuando sea técnicamente posible
// 07

Cambios en esta Política

Esta Política de Privacidad puede actualizarse para reflejar cambios técnicos o mejoras del servicio. La fecha de actualización indicará la versión más reciente.
